# CalcMatxGo

Desktop-приложение на **Golang** для вычислений с матрицами с UI в стиле предоставленного макета: слева список операций, справа ввод/результат.

## Стек

- Go 1.22+
- Fyne (`fyne.io/fyne/v2`) для desktop UI

## Функционал

- Определитель
- Транспонировать
- Ранг
- След
- Возвести в степень
- Умножить на число
- Обратная
- Треугольный вид
- Ступенчатый вид
- LU разложение
- Элементарные преобразования
- Вычислить выражение (`A`, `B`, `+`, `-`, `*`, скобки)

## Архитектура

```text
.
├── cmd
│   └── calcmatxgo
│       └── main.go              # Точка входа desktop-приложения
├── internal
│   ├── matrix
│   │   ├── matrix.go            # Базовые операции, парсинг матриц, trace
│   │   ├── advanced.go          # Determinant, inverse, LU, echelon, power, elementary
│   │   └── matrix_test.go       # Базовые unit-тесты ядра
│   ├── parser
│   │   └── expression.go        # Парсер и вычислитель выражений над матрицами
│   └── ui
│       └── app.go               # Сборка окна/элементов интерфейса и маршрутизация операций
├── go.mod
└── go.sum
```

## Запуск

```bash
go mod tidy
go run ./cmd/calcmatxgo
```

## Примечание

В проекте пока остаются старые веб-файлы (`index.html`, `src/...`) из предыдущей итерации, но основная реализация ТЗ теперь сделана как desktop-приложение на Go.
