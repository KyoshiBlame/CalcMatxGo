# CalcMatxGo

Веб-калькулятор матриц с архитектурой, разделённой на вычислительное ядро и UI-слой.

## Функционал

- Определитель
- Транспонирование
- Ранг
- След
- Возведение матрицы в степень
- Умножение матрицы на число
- Обратная матрица
- Приведение к треугольному виду
- Приведение к ступенчатому виду
- LU-разложение
- Элементарные преобразования строк
- Вычисление выражений над матрицами (`A`, `B`, `+`, `-`, `*`, скобки)

## Архитектура

```text
.
├── index.html              # Страница и контейнеры интерфейса
└── src
    ├── app.js              # Оркестратор UI и вызовов операций
    ├── styles.css          # Стили, включая левое меню как на макете
    ├── core
    │   ├── matrix.js       # Математические операции и валидация
    │   └── parser.js       # Парсинг матриц и выражений
    └── ui
        └── operations.js   # Справочник операций меню
```

## Запуск

Откройте `index.html` в браузере или поднимите простой сервер:

```bash
python3 -m http.server 4173
```

После этого откройте `http://localhost:4173`.
